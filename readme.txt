== Description ==

Change Google Listings & Ads product visibility for defined amount of products at once.

== Installation ==

1. Upload `uv-gla-visibility-changer.zip` using the plugins-page in WordPress or upload the un-zipped folder to `/wp-content/plugins/uv-gla-visibility-changer`.
2. Activate the plugin through the 'Plugins' menu in WordPress
